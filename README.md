# 💰 零花钱系统

AstrBot 插件 —— 给你的 bot 一个完整的虚拟经济生态。

## ✨ 功能一览

**经济核心** — 零花钱余额、存折（审批制取款）、银行（1w起存有利息）

**超市** — 每日刷新，按季节上架应季商品，支持 AI 生成商品。食品有保质期，收藏品永久保存（2%稀有出现率）。节日当天自动上架节日限定。

**小游戏** — 刮刮乐、猜大小、虚拟炒股（10只股票按等级解锁）

**成长系统** — 等级（Lv.0~20）、签到连续加成、22个成就

**背包** — 共享背包 + 用户专属格子，物品有保质期

**跨bot赠送** — 群内多个bot之间互送物品，HMAC签名防伪

**LLM深度集成** — 9个工具让bot自主签到、逛超市、买东西、炒股

## 📋 指令

| 类别 | 指令 |
|------|------|
| 经济 | `签到` `等级` `成就` `银行` `存银行` `取银行` |
| 超市 | `逛超市` `购买 <编号/名称>` |
| 游戏 | `刮刮乐` `猜大 <金额>` `猜小 <金额>` `股市` `买股票` `卖股票` `我的股票` |
| 背包 | `我的格子` |
| 社交 | `发表扬信` `发投诉信` `表扬信排行` |
| 帮助 | `零花钱帮助` |

大部分功能也可以直接和 bot 说话触发（通过 LLM 工具调用）。

管理员指令见 `零花钱帮助`。

## ⚙️ 配置

安装后在 AstrBot 插件配置页面设置：

- **管理员称呼** — bot 会用这个名字称呼你
- **AI生成商品 Provider** — 选择一个已配置的模型，留空用默认商品池
- **Bot名称** — 填写后开启跨bot赠送功能

其他配置按需调整，默认值即可正常使用。

## 📁 项目结构

```
├── main.py              # 插件入口 + 指令 + LLM工具
├── managers/
│   ├── money.py         # 小金库 + 存折
│   ├── backpack.py      # 背包（含保质期）
│   ├── shop.py          # 超市（季节/节日/AI/收藏品）
│   ├── games.py         # 刮刮乐 + 炒股
│   ├── level.py         # 等级 + 签到
│   ├── bank.py          # 银行
│   ├── achievement.py   # 成就
│   ├── gift.py          # 跨bot赠送
│   ├── isolation.py     # 用户隔离池
│   └── thank_letter.py  # 表扬信
```
